## 1. 版本格式

采用语义化版本控制（Semantic Versioning）：

```
主版本号.次版本号.修订号
例如：1.0.0
```

| 级别 | 说明 | 示例 |
|------|------|------|
| 主版本号 | 不兼容的 API 变更 | 1.x.x → 2.x.x |
| 次版本号 | 向后兼容的功能新增 | 1.0.x → 1.1.x |
| 修订号 | 向后兼容的问题修复 | 1.0.0 → 1.0.1 |

## 2. 版本历史

### 1.0.0 (2024-01-15)

#### 新增功能
- ✅ 用户管理系统
  - 用户注册、登录、登出
  - JWT 认证机制
  - RBAC 角色权限模型（Admin/Operator/Viewer）
  - 用户状态管理（激活/禁用）

- ✅ 设备管理系统
  - 设备注册与认证
  - 设备状态监控（在线/离线/离开）
  - 设备审批流程（待审批/已批准/已拒绝）
  - 组织分组管理

- ✅ 许可证管理系统
  - Ed25519 数字签名许可证生成
  - 许可证验证与状态查询
  - 支持 Basic/Pro/Enterprise 三级授权
  - 设备数量限制管理

- ✅ 审计日志系统
  - 多类型日志记录（认证/设备/会话/配置）
  - 条件筛选查询
  - 操作追踪与详情记录

- ✅ Web API 服务
  - RESTful 接口设计
  - CORS 跨域支持
  - 错误处理与状态码规范

- ✅ Docker 容器化部署
  - 多阶段构建优化镜像大小
  - Docker Compose 服务编排
  - Nginx 反向代理配置

- ✅ API 文档
  - OpenAPI 规范文档
  - Swagger UI 可视化界面

#### 修复问题
- 无

- SQLite 数据库查询优化
- 内存缓存策略实现
