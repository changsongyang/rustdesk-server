## 1. 概述

欢迎使用 RustDesk Pro Server！本手册将指导您如何使用系统的各项功能。

**系统特性**：
- 用户认证与授权管理
- 设备注册与审批流程
- 许可证密钥管理
- 操作审计与日志追踪

## 2. 系统登录

### 2.1 访问系统

打开浏览器，访问系统地址：`http://<server-ip>:8080`

### 2.2 登录界面

| 字段 | 说明 | 默认值 |
|------|------|--------|
| 用户名 | 登录账号 | admin |
| 密码 | 登录密码 | admin123 |

**首次登录建议**：立即修改默认密码，确保系统安全。

### 2.3 登录流程

1. 输入用户名和密码
2. 点击「登录」按钮
3. 系统验证凭据
4. 成功后跳转到仪表盘

### 2.4 密码重置

如果忘记密码，请联系管理员重置密码。

## 3. 用户管理

### 3.1 用户列表

登录后，点击左侧菜单「用户管理」进入用户列表页面。

**列表功能**：
- 查看所有用户
- 按角色筛选
- 按状态筛选（激活/禁用）
- 分页浏览

### 3.2 创建用户

1. 点击「创建用户」按钮
2. 填写用户信息：
   - **用户名**：唯一标识，3-64 字符
   - **邮箱**：有效邮箱地址
   - **密码**：至少 8 字符
   - **角色**：选择用户角色
3. 点击「保存」

### 3.3 用户角色

| 角色 | 权限说明 |
|------|----------|
| 管理员 | 完整系统权限 |
| 操作员 | 设备管理、会话管理 |
| 查看者 | 只读权限，不可修改 |

### 3.4 编辑用户

1. 在用户列表中点击用户名称
2. 修改用户信息（用户名不可修改）
3. 点击「保存」

### 3.5 禁用/启用用户

1. 在用户列表中找到目标用户
2. 点击「禁用」或「启用」按钮
3. 确认操作

### 3.6 删除用户

> **注意**：删除用户将同时删除其关联的设备和会话记录。

1. 在用户列表中找到目标用户
2. 点击「删除」按钮
3. 确认删除操作

## 4. 设备管理

### 4.1 设备列表

点击左侧菜单「设备管理」进入设备列表页面。

**设备状态**：
| 状态 | 图标 | 说明 |
|------|------|------|
| 在线 | 🟢 | 设备当前在线 |
| 离线 | 🔴 | 设备离线 |
| 离开 | 🟡 | 设备离开状态 |

**审批状态**：
| 状态 | 说明 |
|------|------|
| 待审批 | 设备等待管理员审批 |
| 已批准 | 设备已通过审批 |
| 已拒绝 | 设备未通过审批 |

### 4.2 设备审批

1. 在设备列表中找到待审批设备
2. 点击设备名称查看详情
3. 点击「批准」或「拒绝」按钮
4. 添加审批备注（可选）

### 4.3 查看设备详情

1. 点击设备名称进入详情页
2. 查看设备信息：
   - 设备名称
   - 设备 ID
   - 所属用户
   - 在线状态
   - 注册时间
   - 最后在线时间

### 4.4 删除设备

1. 在设备列表中找到目标设备
2. 点击「删除」按钮
3. 确认删除操作

### 4.5 搜索设备

使用搜索框按设备名称或设备 ID 搜索设备。

## 5. 许可证管理

### 5.1 许可证状态

点击左侧菜单「许可证管理」查看当前许可证状态。

**许可证信息**：
- 许可证类型
- 最大设备数
- 已使用设备数
- 有效期截止日期
- 剩余天数

### 5.2 生成许可证

1. 点击「生成许可证」按钮
2. 选择许可证类型：
   - **基础版**：适合小型团队
   - **专业版**：适合中型企业
   - **企业版**：适合大型组织
3. 设置最大设备数
4. 设置有效期（天数）
5. 点击「生成」

### 5.3 许可证密钥格式

```
RD-{TYPE}-{RANDOM}-{RANDOM}-{RANDOM}
示例：RD-PRO-ABCDE-FGHIJ-KLMNO
```

### 5.4 验证许可证

1. 点击「验证许可证」按钮
2. 输入许可证密钥
3. 点击「验证」
4. 查看验证结果

## 6. 审计日志

### 6.1 日志列表

点击左侧菜单「审计日志」查看操作日志。

**日志类型**：
| 类型 | 说明 |
|------|------|
| 登录 | 用户登录系统 |
| 登出 | 用户退出系统 |
| 用户创建 | 创建新用户 |
| 用户更新 | 更新用户信息 |
| 用户删除 | 删除用户 |
| 设备注册 | 新设备注册 |
| 设备审批 | 设备审批操作 |
| 设备删除 | 删除设备 |
| 许可证生成 | 生成许可证 |
| 许可证验证 | 验证许可证 |
| 配置更新 | 更新系统配置 |

### 6.2 筛选日志

- **按类型筛选**：选择日志类型
- **按用户筛选**：选择操作用户
- **按时间范围筛选**：设置开始时间和结束时间

### 6.3 查看日志详情

1. 点击日志记录查看详情
2. 查看：
   - 操作类型
   - 操作用户
   - 目标资源
   - 详细信息
   - IP 地址
   - 操作时间

## 7. 系统设置

### 7.1 基本设置

点击左侧菜单「系统设置」→「基本设置」

**设置项**：
- 系统名称
- 系统描述
- 时区设置
- 语言设置

### 7.2 安全设置

点击「安全设置」

**设置项**：
- 密码策略（最小长度、复杂度要求）
- 登录失败锁定（尝试次数、锁定时间）
- 会话超时时间
- IP 白名单

### 7.3 邮件设置

点击「邮件设置」

**设置项**：
- SMTP 服务器地址
- SMTP 端口
- 用户名
- 密码
- 发件人邮箱

### 7.4 备份设置

点击「备份设置」

**设置项**：
- 自动备份开关
- 备份时间
- 备份保留天数
- 备份路径

## 附录：常见问题

### Q1：忘记密码怎么办？

联系系统管理员重置密码。

### Q2：设备无法注册怎么办？

1. 检查设备网络连接
2. 确认服务器地址正确
3. 检查设备是否已被拒绝
4. 联系管理员查看日志

### Q3：许可证过期怎么办？

1. 生成新的许可证
2. 在系统中更新许可证
3. 联系销售团队续费

### Q4：如何导出数据？

在对应列表页面寻找「导出」按钮，支持 Excel 格式导出。
